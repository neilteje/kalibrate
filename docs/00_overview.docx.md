# CS498 AI Agents in the Wild \- Final Project Overview

## Project Overview

In this course, you will design, implement, and evaluate an AI agent system. Your project will culminate in **two separate papers**: one documenting your benchmark design and another documenting your agent implementation.

## Key Requirements

* Design and implement an AI agent for a specific task domain

  * Implement your *own* agent.

  * Do not use existing full agent implementations or frameworks  (e.g., LangGraph) as your solution. Standard libraries/APIs are fine, but the agent logic and orchestration must be your own.

* Create a custom benchmark (per team):

  * The benchmark must be independent of your agent, so others could use it to evaluate their own agents.

  * From scratch: 10 tasks (3-credit) or 20 tasks (4-credit). If you reuse an existing benchmark structure/framework, you must double the number of tasks (20 / 40\) and clearly explain what is meaningfully different. In your benchmark paper, state whether you used an existing framework (and which one).

  * Mixed-credit teams follow the 4-credit requirement.

* **No coding/programming agents** \- choose a different domain

* Write **two separate papers**:

  * **Benchmark paper** (3-4 pages): Document your task suite design

  * **Agent paper** (3-4 pages): Document your agent implementation and evaluation

* Present your findings to the class

## Project Ideas (Examples)

* **Multi-turn customer service agent** with context retention and escalation handling

* **Research assistant agent** for literature search and summarization

* **Game-playing agent** for strategy games requiring long-term planning

* **Travel planning agent** handling dynamic constraints and user preference changes

* **Educational tutoring agent** adapting to student understanding

* **Household task planning agent** for complex multi-step chores

* **Meeting scheduling agent** negotiating across multiple participants

* **Recipe adaptation agent** handling dietary restrictions and ingredient substitutions
